# Codex UI: return to conversation-style view

If you are seeing the large green/red diff pane, you are in **task diff mode**.

To get back to chat-style flow:
1. Click the **X** at the top-right of the diff pane.
2. In the left panel, continue in the message box (chat stays in same task).
3. If it keeps reopening, open a normal ChatGPT thread (not the `/codex/cloud/tasks/...` page) for conversation-only mode.

Note: Any new code commit in task mode can reopen the diff panel.
